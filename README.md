# Algorithm

## **선택정렬**

-i(0~N-1)번 인덱스부터 순차적으로 가장 작은 값이라 가정하고 i+1번째 인덱스부터 N-1번째 인덱스까지 
 i번째 인덱스보다 작은 값이 있는지 확인하여 있다면 제일 작으값으로 교환(오른차순 기준)
